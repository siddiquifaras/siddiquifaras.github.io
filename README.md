# siddiquifaras.github.io

Personal portfolio website for Faras Siddiqui - Founding Engineer at CVision.

## Live Site

Visit: [https://siddiquifaras.github.io](https://siddiquifaras.github.io)

## Structure

```
├── index.html          # Main landing page
├── bio.html            # Detailed bio and skills
├── education.html      # Education background
├── industry.html       # Work experience
├── pet-projects.html   # Personal projects
├── publications.html   # Research publications
├── 404.html            # Custom 404 page
├── styles.css          # All styling
├── script.js           # JavaScript interactions
├── favicon.svg         # Site favicon
├── robots.txt          # SEO robots file
├── sitemap.xml         # SEO sitemap
├── .nojekyll           # Disable Jekyll processing
├── images/             # Profile photo and assets
└── files/              # CV and downloadable files
```

## Setup

1. Add your profile photo as `images/profile.jpeg` ✓
2. Add your CV as `files/Faras_Siddiqui_CV_AI_Engineer.pdf` ✓
3. Push to GitHub and enable GitHub Pages

## Local Preview

Simply open `index.html` in your browser.
